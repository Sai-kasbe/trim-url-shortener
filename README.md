# Trim URL Shortener

A full-stack URL shortener built with:

- React (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)

## Features

- Create short URLs
- Custom aliases
- Redirect to original URLs
- Click tracking
- Analytics dashboard
- Device breakdown

## Setup

### Backend

```bash
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Tech Stack

- React
- Vite
- Node.js
- Express
- MongoDB
- Mongoose
